Allows you to easily inline javascript into posts and apply css to a div wrapper.  Specifically designed for Infusionsoft Hosted Webforms.

== Description ==

This plugin provides a "javascript" short code.  If you specify the src url from the javascript tag provided by Infusionsoft in the webform "Publish" tab for the src attribute of the shortcode it will plop your webform into post or page.

== Installation ==

No installation required.
